# zhyx1996.github.io

这是一个基于 GitHub Pages 的个人主页仓库，用于展示个人简介、公开仓库、Star 项目、博客文章，以及部分公开信息嵌入示例。

## 项目内容

当前站点主要包含以下页面：

- `index.html`：主页，总览个人方向、近期关注、文章摘要、市场信息和公开展示内容。
- `projects.html`：公开仓库展示页。
- `stars.html`：GitHub Stars 展示页。
- `articles.html`：博客园文章展示页。
- `public-data-proxy/embed-demo.html`：公开嵌入方式演示页。
- `app.js`：页面数据渲染与前端交互逻辑。
- `styles.css`：全站样式文件。

## 技术栈

- 原生 HTML
- 原生 CSS
- 原生 JavaScript
- GitHub Pages

仓库不依赖前端构建工具，页面可直接以静态文件形式部署。

## 数据来源

站点中的展示内容主要来自以下几类公开信息：

- GitHub 个人资料、公开仓库与 Stars
- 博客园公开文章
- 部分第三方公开页面或官方嵌入组件

为保证页面在接口不可用时仍能展示基础内容，前端代码中保留了部分兜底数据。

## 本地预览

这是一个纯静态站点，克隆后可直接用任意静态文件服务器预览。例如：

```bash
python -m http.server 8000
```

然后在浏览器中访问：

```text
http://localhost:8000
```

> 建议通过本地静态服务器访问，而不是直接双击 HTML 文件，以避免部分资源路径或网络请求行为与部署环境不一致。

## 校验

仓库当前没有完整的构建、测试或 lint 流程，修改 JavaScript 后可先执行：

```bash
node --check app.js
```

## 部署

本仓库面向 GitHub Pages 部署，提交到默认分支后即可由 Pages 发布静态页面。

## 维护建议

- 新增页面时尽量沿用现有导航与视觉风格。
- 修改 `app.js` 中的数据获取逻辑时，注意保留兜底数据与异常处理。
- 如果更新外部公开入口，请同步检查对应页面中的按钮链接与文案。
